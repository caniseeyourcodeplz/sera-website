## PHASE 2: STRATEGIC REFINEMENT

### Step 5: Refine Critical Sections

Start with high-impact areas. Use this approach for each:

**For Each Key Section (Hero, Benefits, Social Proof, Pricing, CTA):**

```
/impeccable refine [section-name] + 

"This section currently [current state]. I want it to:
- Use the healthcare green + charcoal color palette (no [current colors])
- Feel professional and trustworthy (not playful/casual)
- Have clear visual hierarchy
- Include strong outcome-focused copy
- Show [specific element] more prominently

Show me 3 variants with different approaches. 
Make them visually distinct and compare-able."
```

**Example - Hero Section:**
```
/impeccable refine hero + 

"The hero currently feels [unclear/generic/too flashy]. I want it to:
- Lead with the pain point: 'Missed calls are killing your revenue'
- Use charcoal background with healthcare green accent
- Feature a clear CTA: 'Book Your Demo' (primary), 'See How It Works' (secondary)
- Include a dashboard mockup showing Sera working
- Build trust with a short credibility statement

Show me 3 distinct hero approaches. 
Option 1: Problem-focused (dark, serious)
Option 2: Visual-focused (dashboard prominent)
Option 3: Data-focused (ROI statistics visible)

Make them dramatically different so I can see which direction resonates."
```

**Example - Benefits Section:**
```
/impeccable refine benefits + 

"Current benefits section feels scattered and feature-focused. I want it to:
- Focus on OUTCOMES not features: Revenue, Operations, Patient Experience
- Use clean cards with healthcare green accents
- Replace generic icons with specific benefit visuals
- Lead with numbers/metrics where possible
- Be scannable (clinic owners are busy)

Create 3 layout variants:
Option 1: Vertical stack (mobile-first)
Option 2: 3-column grid (desktop focus)
Option 3: Hybrid (staggered, visually interesting)

Make sure the copy emphasizes ROI and time-saved."
```

**Example - Social Proof:**
```
/impeccable refine testimonials + 

"Testimonials currently lack specificity and numbers. I want them to:
- Include REAL metrics: 'Recovered $40K in revenue', 'Reduced missed calls 87%'
- Feature clinic name, location, and owner name (builds trust)
- Be outcome-focused: 'Our team now has X hours back'
- Look professional, not generic review-site style
- Include a subtle clinic logo or image if available

Create 3 testimonial card designs that feel premium and trustworthy."
```

**Example - Pricing Section:**
```
/impeccable refine pricing + 

"Pricing currently feels [unclear/aggressive/too complex]. I want it to:
- Show 2-3 clear tiers with transparent pricing
- Include specific features per tier (not vague)
- Lead with ROI: 'Average clinic saves $X with Sera'
- Have a strong CTA: 'Start Free Trial' or 'Book Demo'
- Feel fair and value-based, not predatory

Create 3 pricing layouts:
Option 1: Horizontal cards (price comparison)
Option 2: Table format (feature comparison)
Option 3: Tiered staircase (premium focus)

All should feel trustworthy and transparent."
```

## PHASE 3: USE IMPECCABLE LIVE MODE FOR FINE-TUNING

- Click on any component
- Use pre-built commands like `bolder`, `trust`, `delight`, `quieter`
- Create variants of specific sections
- Tune colors, spacing, typography at micro-level
- Preview changes in real-time

**Key Commands to Use in Live Mode:**


## CRITICAL DO's AND DON'Ts

### ❌ DON'T (Current Problems to Avoid)
- Generic clinic stock photos
- Bright, playful colors (healthcare = serious)
- Overused SaaS layouts (bento boxes, minimal cards)
- Flashing animations or dramatic motion
- Ambiguous CTAs ("Learn More", "Click Here")
- Multiple CTAs with equal weight
- Unexplained medical jargon
- Hidden pricing (red flag for B2B)
- Feature-focused copy (not outcome-focused)

### ✅ DO (What We're Building)
- Professional medical imagery (dashboards, data, real clinics)
- Charcoal + Healthcare Green color scheme (trust + growth)
- Clean, spacious layouts with breathing room
- Smooth, purposeful interactions
- Clear, action-oriented CTAs ("Book Demo", "Start Trial")
- Single primary CTA per section
- Plain language explanations
- Transparent, value-based pricing
- Outcome-focused messaging (revenue, time saved, peace of mind)
